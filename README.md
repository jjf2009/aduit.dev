# Resulyze

AI-powered resume feedback for job seekers.

Resulyze helps students and early-career candidates understand how well their resume matches a specific job description. Upload a PDF resume, paste the job description, and get a clear ATS-style score with practical suggestions to improve the resume before applying.

## What It Does

- Checks a resume against a real job description
- Generates an ATS compatibility score
- Finds missing keywords from the job description
- Highlights keywords already matched in the resume
- Gives section-by-section feedback for summary, experience, skills, and education
- Provides actionable rewrite suggestions users can apply immediately

## Who It Is For

Resulyze is built for:

- Students applying for internships
- New graduates applying for entry-level roles
- Career switchers tailoring resumes for a new field
- Anyone who wants faster, clearer resume feedback before submitting an application

## How It Works

1. Upload your resume as a PDF.
2. Paste the job description for the role you want.
3. Click **Analyze Resume**.
4. Review your score, missing keywords, matched keywords, and improvement checklist.
5. Update your resume and apply with more confidence.

## What You Get

Each analysis returns:

- A resume score out of 100
- A short explanation of the score
- Missing job keywords to consider adding
- Keywords your resume already covers
- Feedback by resume section
- A prioritized checklist of improvements
- An overall verdict: strong, moderate, or weak

## Privacy Note

Resume analysis happens only when a user submits a PDF and job description. The current MVP does not include user accounts, resume storage, payments, or a database.

## Product Status

Resulyze is currently an MVP focused on one core workflow:

```text
Upload PDF + Paste Job Description -> Analyze -> Improve Resume
```

The goal is simple: give job seekers fast, useful, and easy-to-understand resume feedback.
